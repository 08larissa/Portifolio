Sobre

Sou apaixonada por programação e amo transformar ideias em experiências interativas e divertidas. Meu foco é explorar diferentes tecnologias para criar soluções criativas e funcionais. Estou sempre em busca de novos desafios e aprendizados, porque acredito que a evolução no desenvolvimento vem da curiosidade e da prática constante.

Tenho experiência com projetos criativos usando p5.js e outras tecnologias web. Adoro criar jogos simples e experiências interativas que engajam os usuários.

Habilidades

HTML

Linguagem de marcação para estruturar o conteúdo de páginas web, definindo elementos como textos, imagens e links.

CSS

Linguagem de estilização que controla a aparência visual dos elementos HTML, permitindo criar layouts responsivos e animações.

JavaScript

Linguagem de programação que adiciona interatividade às páginas web, permitindo criar aplicações dinâmicas e manipular elementos em tempo real.

Scratch

Plataforma de programação visual que permite criar jogos, animações e histórias interativas através de blocos de código.
